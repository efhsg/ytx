# Skill Routing

When working on specific file types or topics, proactively load the relevant skill.

## File Pattern Routing

| When editing | Load skill |
|--------------|------------|
| `ytx.py` | `rules/coding-standards.md` |
| `tests/*` | `rules/testing.md` |
| `.claude/skills/*` | `skills/improve-prompt.md` |
| `.claude/commands/*` | `skills/index.md` |

## Topic Routing

| When discussing | Load skill |
|-----------------|------------|
| Refactoring, restructuring | `skills/refactor.md` |
| Code review, PR review | `skills/review-changes.md` |
| New feature branch | `skills/new-branch.md` |
| Prompt improvement | `skills/improve-prompt.md` |
| Testing, pytest, mocking | `rules/testing.md` |
